# 5 套风格候选

> 说明：本次测试没有调用外部图片生成接口保存位图；为保证效果测试有可视化候选，已在同目录生成 5 张 SVG mini style board。每套也给出可直接用于生图的 prompt 和验收说明。

## 1. 业务冷静型

- 候选图：`style-board-business-calm.svg`
- 画面说明：浅灰背景、深色正文、绿色强调线、克制卡片和流程节点，适合公司内部复盘或管理层说明。
- 配色：雾灰、墨黑、冷绿、少量橙色提醒。
- 字体感：现代无衬线，字重清楚，信息密度中高。
- 图表气质：条形图、流程线、对比卡片，强调可读性。
- 透明素材使用方式：少量线性图标、箭头、流程节点。
- 生图 prompt：`Create a mini style board for a calm business presentation about turning an idea into a deliverable visual plan with Codex. Light warm gray background, dark editorial typography, restrained green accent line, subtle orange highlights, clean process cards, editable chart feeling, no fake text, no logos, 16:9.`
- 验收说明：第一眼应像“能拿去开会”的方案页，不应像营销海报。

## 2. 编辑杂志型

- 候选图：`style-board-editorial.svg`
- 画面说明：大标题、版面留白、图片占位与文本块错落，像一篇视觉专栏被整理成 PPT。
- 配色：象牙白、深炭黑、番茄红、青蓝点缀。
- 字体感：标题有杂志感，正文清爽。
- 图表气质：少量信息图，不抢叙事。
- 透明素材使用方式：手写箭头、贴纸式标签、轻量插画元素。
- 生图 prompt：`Create a 16:9 editorial magazine style board for a presentation: ordinary people use Codex to turn an idea into a visual deliverable. Large confident Chinese-friendly title area, generous white space, image placeholders, tomato red and teal accents, tactile paper texture, no readable fake text, no logos.`
- 验收说明：应有“课程开场”的吸引力，同时保持正文空间。

## 3. 产品演示型

- 候选图：`style-board-product-demo.svg`
- 画面说明：界面窗口、步骤面板、输出物预览，像 SaaS 产品发布或工具教学。
- 配色：白底、深蓝灰、亮青、柠檬黄小面积提示。
- 字体感：清晰、数字产品感强。
- 图表气质：流程图、状态标签、任务列表。
- 透明素材使用方式：窗口、按钮、光标、输出文件图标。
- 生图 prompt：`Create a 16:9 product demo style board showing Codex workflow from idea to visual deliverables. Clean SaaS interface panels, command workspace, task checklist, preview cards for PPTX and visual assets, cyan accents, small lemon highlights, no real brand logos, no readable fake UI text.`
- 验收说明：应让用户立刻理解“这是一个可操作的工作台”。

## 4. 数据图表型

- 候选图：`style-board-data-report.svg`
- 画面说明：报告感更强，用图表和指标解释流程价值，适合咨询、投研、趋势报告。
- 配色：冷白、蓝黑、紫红、薄荷绿。
- 字体感：理性、严谨、偏报告。
- 图表气质：条形图、矩阵、成熟度曲线。
- 透明素材使用方式：少量图表注释和指标徽标。
- 生图 prompt：`Create a 16:9 data report style board for explaining how Codex converts ideas into visual deliverables. Precise dashboard composition, editable chart feeling, matrix blocks, dark navy typography, mint and magenta accents, clean grid, no fake readable text, no logos.`
- 验收说明：适合强调“效率、质量、可复用流程”，不适合太感性故事。

## 5. 视觉叙事型

- 候选图：`style-board-visual-story.svg`
- 画面说明：把“想法”画成一个从便签到交付包的旅程，适合 AI 通识课和创意提案。
- 配色：奶白、石墨黑、天空蓝、珊瑚粉、嫩绿。
- 字体感：亲和、有手作感，但不幼稚。
- 图表气质：路线图、故事分镜、阶段徽章。
- 透明素材使用方式：便签、灯泡、画板、文件包、路线箭头。
- 生图 prompt：`Create a 16:9 visual storytelling style board: an ordinary person's idea travels from a sticky note to a polished visual delivery package with Codex. Friendly editorial illustration, clean white background, sky blue, coral and soft green accents, journey map, transparent sticker-like assets, no readable fake text, no logos.`
- 验收说明：应有故事感，但不能变成儿童插画。

## 最终选择

选择 2「编辑杂志型」和 3「产品演示型」混合：用杂志式开场建立吸引力，用产品演示式中段讲清操作路径。最终 deck spec 采用浅底、深色字、青蓝主强调、番茄红辅助强调。
